# PR24 Outcomes Risk Modelling – R Code and Supplementary Files

This repository contains R code and supplementary files that support Ofwat’s PR24 Outcomes risk modelling.

These files are provided alongside the **PR24 Outcomes Risk Modelling Guide** and are intended to help users understand, replicate, or test elements of the modelling approach described in that guide. Further detail on the methodology, assumptions, and how these files are used can be found in the guide.

## Contents

This repository includes:

- R code used to support elements of the risk modelling approach (for example, anchoring forecasts and Performance Commitment Levels (PCLs)).
- Supplementary files used as helper inputs and outputs alongside the main Excel risk models.

These files are not published on the Ofwat website but are provided here to improve transparency and accessibility.

## Relationship to published models

The primary risk models (Excel) referenced in the PR24 Outcomes Risk Modelling Guide are published on the Ofwat Website. 
https://www.ofwat.gov.uk/regulated-companies/price-review/2024-price-review/final-determinations-models/#outcomesmodels 
This repository complements those published models by providing supporting code and supplementary files used in the modelling process.

Unless otherwise stated, files in this repository reflect the **PR24 Final Determinations** position. Users may adapt the code or files if they wish to test alternative assumptions or scenarios.

## Use of the files

These materials are provided to support understanding of Ofwat’s approach to outcomes risk modelling. Users should refer to the Risk Modelling Guide for full context, explanations of assumptions, and guidance on appropriate use.
